# 📖 MANUAL DE INSTALACIÓN CERO-ERRORES: VANTEC VCORE L6
**Objetivo:** Instalar el sistema en un servidor nuevo en menos de 5 minutos, garantizando licencias y rutas relativas.

---

## 🏗️ FASE 1: EMPAQUETADO (En tu PC de Desarrollo)
*Regla de Oro: Nunca copies la carpeta de desarrollo tal cual al cliente. Lleva basura.*

1. Abre tu carpeta de desarrollo actual.
2. Ejecuta el archivo: `00_Empaquetador_Build_Release_VCore.bat`.
3. El sistema creará automáticamente una carpeta nueva dentro de tu proyecto llamada **`VCore_Release_v6.3.4_GOLD`**.
4. Copia esa carpeta en una Memoria USB. Esta es la que viajará al Servidor Nuevo.

---

## 🛠️ FASE 2: PREPARACIÓN DEL SERVIDOR NUEVO
*Enciende el servidor del cliente y asegúrate de instalar estos dos motores.*

### 2.1 Instalar Python (3.12.7)
1. Ejecuta el instalador de Python como **Administrador**.
2. **CRÍTICO (Paso para Dummies):** En la primera pantalla, abajo del todo, marca la casilla que dice **"Add Python to PATH"**. Si no marcas esto, nada funcionará.
3. Haz clic en "Install Now".
4. Al terminar, si sale un botón que dice **"Disable path length limit"**, hazle clic. Luego cierra.

### 2.2 Instalar PostgreSQL (17 o 18)
1. Ejecuta el instalador de Postgres.
2. Cuando te pida la contraseña, escribe: `postgres` (en minúsculas).
3. Dale "Siguiente" a todo lo demás hasta que termine.

---

## 🚀 FASE 3: INSTALACIÓN EN 1 CLIC
*Ya puedes pegar la carpeta donde quieras, pero sugerimos la raíz de C:\.*

1. Conecta tu USB al Servidor.
2. Pega la carpeta `VCore_Release_v6.3.4_GOLD` en `C:\`.
3. Renombra la carpeta a `gestor_vcore` (Ruta final sugerida: `C:\gestor_vcore`).
4. Abre la carpeta y **ejecuta como Administrador el archivo `install_vcore.bat`**.

**¿Qué hace este script solo?**
* Crea la carpeta `\license` y extrae el ADN del servidor (`machine_id.txt`).
* Crea la Base de Datos (`vcore_blank_db`).
* Instala todas las librerías automáticamente.
* Genera las rutas relativas en el archivo `.env`.

---

## 🔐 FASE 4: ARRANQUE Y ACCESO MAESTRO
1. Dentro de la carpeta, ejecuta: `start_vcore.bat`.
2. Verás que se abren dos pantallas negras.
3. **Inyectar Usuario:** Ejecuta una vez el archivo `seed_admin.py` (hazle doble clic).
4. Abre el navegador web y entra a: `http://localhost:8000`
5. **Credenciales Maestras:**
    * Usuario: `admin`
    * Contraseña: `admin123`
6. Si ves el botón de "Empresas", ¡felicidades! El sistema ya es tuyo.

---

## 🚨 FASE 5: LIMPIEZA DE SEGURIDAD (OBLIGATORIO)
*Este es el paso más importante para que nadie te robe el acceso.*

1. **Borrar la Llave:** Busca el archivo llamado `seed_admin.py` dentro de tu carpeta `gestor_vcore` y **BORRALO**. 
2. **¿Por qué?** Porque ese archivo es una "llave maestra". Si alguien entra a tu computadora y lo usa, podrá entrar a tu sistema sin permiso.
3. **Cambiar Contraseña:** Entra a tu perfil en el sistema y cambia la clave `admin123` por una que solo tú sepas.

---

## 🔑 ¿Qué pasa cuando el cliente paga la licencia?
1. Ve a la carpeta `C:\gestor_vcore\license`.
2. Envía el archivo `machine_id.txt` a soporte de Vantec.
3. Te regresarán un archivo de licencia. Pégalo en esa misma carpeta y reinicia el sistema.
